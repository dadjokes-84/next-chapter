# Next-Chapter 💕

**Dating app for single parents. Affordable. Ad-free. Built for real life.**

## About

Next-Chapter is a mobile-friendly dating app designed specifically for single parents. At just $5.99/month (or free with limits), it's the most affordable option in the market — and the only one with zero ads.

## Features

- 🔍 **Smart Matching** — Find compatible single parents in your area
- ✅ **Verified Profiles** — Selfie verification for safety
- 💬 **Real Conversations** — Message unlimited matches on premium
- 🔒 **Privacy First** — No ads, no tracking nonsense
- 📍 **Local Focus** — Start with Illinois, expand nationwide

## Tech Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** PostgreSQL (Supabase)
- **Payments:** Stripe
- **Hosting:** Vercel

## Getting Started

### Prerequisites
- Node.js 16+
- PostgreSQL (or Supabase account)
- Stripe account (test keys)

### Setup

```bash
# Clone repo
git clone https://github.com/dadjokes-84/next-chapter.git
cd next-chapter

# Frontend setup
cd frontend
npm install
npm run dev

# Backend setup (in another terminal)
cd backend
npm install
npm run dev
```
